# Sunday Hikers Data Pipeline

## Overview
This project aims to build an automated data pipeline that extracts data from the
Sunday Hikers website, stores it in a database, and visualizes it using Power BI.

## Data Source
- WordPress REST API
- Page: "Past Hikes"

## Planned Architecture
WordPress API → Python → Database → Power BI

## Current Status

- Repository setup
----------------------------------------------
- Data extraction from WordPress
- HTML parsing
- Year-wise event classification
- Data stored in SQLite for local validation
